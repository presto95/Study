### math 모듈

수학과 관련된 다양한 기능을 제공함

```python
import math

math.ceil(3.14)		# 4
math.fabs(-3.14)	# 3.14
math.factorial(3)	# 6
math.floor(3.78)	# 3
math.sqrt(25)		# 5
math.pow(3, 3)		# 27
```

### 클래스

```python
class Car:
    # 생성자 정의 방법
    def __init__(self, num):
        # 클래스 내에 멤버 변수를 따로 선언하지 않아도 된다
        self.horsePower = num
    
    def getHorsePower(self):
        return self.horsePower
# 메소드 선언시 self가 인자로서 자동으로 넘어옴. 해당 자리에 매개변수 생략 가능

myCar = Car(2000)
myCar.getHorsePower	# 2000
```
